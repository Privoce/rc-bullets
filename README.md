# react-barrage

> awesome barrage for react component 

[![NPM](https://img.shields.io/npm/v/react-barrage.svg)](https://www.npmjs.com/package/react-barrage) [![JavaScript Style Guide](https://img.shields.io/badge/code_style-standard-brightgreen.svg)](https://standardjs.com)

## Install

```bash
npm install --save react-barrage
```

## Usage

```jsx
import React, { Component } from 'react'

import MyComponent from 'react-barrage'

class Example extends Component {
  render () {
    return (
      <MyComponent />
    )
  }
}
```
